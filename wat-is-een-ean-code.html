Wat is een EAN-code?

Onder vrijwel elke streepjescode staat een rij cijfers: de EAN-code. Die code is het wereldwijde paspoort van een product. Wie de code kent, weet precies over welk artikel het gaat, tot en met de kleur en de inhoudsmaat. Daarom is de EAN-code de meest betrouwbare manier om prijzen te vergelijken.

De opbouw van de code

EAN staat voor European Article Number. De meest gebruikte variant is de EAN-13, met dertien cijfers die elk een betekenis hebben:

  • De eerste twee of drie cijfers verwijzen naar de GS1-organisatie waar de fabrikant is aangesloten. Een code die met 87 begint, is bijvoorbeeld uitgegeven in Nederland.
  • Daarna volgt het bedrijfsnummer van de fabrikant of importeur.
  • Vervolgens het artikelnummer dat het bedrijf zelf aan het product heeft gegeven.
  • Het laatste cijfer is een controlecijfer.

Op kleine verpakkingen, zoals een pakje kauwgom, zie je soms de korte variant met acht cijfers: de EAN-8. In Noord-Amerika wordt vaak de UPC-code met twaalf cijfers gebruikt, die binnen hetzelfde systeem past. De overkoepelende naam voor al deze codes is GTIN.

Het controlecijfer

Het laatste cijfer van de code is een rekenkundige controle op de overige cijfers. De cijfers worden om en om met 1 en met 3 vermenigvuldigd en opgeteld; het controlecijfer vult die som aan tot het eerstvolgende tiental. Kassasystemen en scanners gebruiken dit om leesfouten te onderscheppen. CompariScan doet dezelfde controle wanneer je een code handmatig intypt, zodat een tikfout meteen opvalt.

Waarom de EAN-code ideaal is voor prijsvergelijken

Productnamen verschillen per winkel. De ene webshop schrijft de uitvoering voluit, de andere kort hem af en een derde voegt er een eigen omschrijving aan toe. De EAN-code heeft daar geen last van: hetzelfde product heeft overal dezelfde code. Scan je de barcode met CompariScan, dan weet je zeker dat je appels met appels vergelijkt.

Waar vind je de code?

De code staat vrijwel altijd onder de streepjescode op de verpakking, vaak op de achterkant of de onderkant. Bij boeken is het ISBN tegelijk de EAN-code. Geen verpakking bij de hand? Maak dan gewoon een foto van het product zelf, dan herkent CompariScan het op het uiterlijk.
